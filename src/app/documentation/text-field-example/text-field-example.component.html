Text Field

A text fields are useful when the form value is expected to be some...you guessed it! Text.


            <af-form 
              [formDefinition]="myFormDefinition" 
              [(formData)]="myFormData" 
            </af-form>
        
        
          import {{'{'}} Component {{'}'}} from '@angular/core';

          @Component({{'{'}}
            selector: 'text-example',
            templateUrl: './text-field-example.component.html'
          })
          export class TextExampleComponent {{'{'}}

            myFormData = {{'{'}}
            textField: 'Some text here'
            {{'}'}};

            myFormDefinition = {{'{'}}
              layout: [['textField']],
              fields: [
                {{'{'}}
                  name: 'textField',
                  fieldType: 'Text',
                  label: 'Text Field Label'
                {{'}'}}
              ]
            {{'}'}}
          {{'}'}}

Validation

There are several validators that can be automatically applied to text fields. Try changing the values of each field to see how the validation messages are change. Check out the documentation for form validation for more details.

              [formDefinition]="myValidatedFormDefinition" 
              [(formData)]="myValidatedFormData" 
            </af-form>
