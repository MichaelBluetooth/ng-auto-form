NFPA Field

An NFPA field displays an NFPA diamond and allows users to select values for the flammability, health, instability (sometimes known as "reactivity") and special codes.

              [formDefinition]="myFormDefinition" 
              [(formData)]="myFormData" 
            </af-form>
