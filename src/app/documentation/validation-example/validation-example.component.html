Validation

This sectiond describes how you can optionally add validation to form controls. It also provides an example of how you can get the state of the forms validity, which can be used to do things like enable/disable buttons and prevent form submission.

Example

There are quite a few different validations that are supported. Try entering valid values for all controls and observe how the button becomes enabled.


<af-form 
  [formDefinition]="myValidatedFormDefinition" 
  [(formData)]="myValidatedFormData" 
</af-form>
<button [disabled]="!isFormValid">Print Form Data</button>
