CASNumber Field

A CAS Number field is a glorified text field that will display a warning message when the input isn't a valid CAS Number. The control will still be considered valid, even if the CAS No is invalid.

Try entering a valid CAS No to see the warning dissapear. Some example values are: 7732-18-5, 50-36-2
