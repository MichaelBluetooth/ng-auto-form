MultiList Field

MultiLists are used when the user is expected to select one or more values from a constrained set of possible values.

Like list fields, for simple multi list options, an array of strings can be provided.

When the value of the option does not match what needs to be displayed, you can provide a list of objects. If the list options are objects, by default the control will look for 'id' and 'name' fields. Alternatively you can specify which fields are used for the value and display.


            <af-form 
              [formDefinition]="myFormDefinition" 
              [(formData)]="myFormData" 
            </af-form>
